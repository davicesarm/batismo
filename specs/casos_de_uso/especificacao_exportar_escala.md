# Caso de Uso: Exportar Escala

**1. Identificação**
**Nome:** Exportar Escala
**Ator Principal:** Usuario

**2. Fluxo Principal**
1. O ator acessa a lista de escala.
2. O ator seleciona o formato de saída: PDF, Planilha (Excel) ou Texto.
3. O sistema gera o arquivo e inicia o download ou compartilhamento.

**3. Regras de Negócio**
* **RN01:** O arquivo deve conter: Data, Hora, Local, Celebrante e Casal Responsável.
