# Batismo

Este é um projeto web, cujo frontend é desenvolvido com [Next.js](https://nextjs.org) e seu respectivo [backend em Spring Boot](https://github.com/davicesarm/batismo-api).

## Descrição

O sistema "Batismo" é uma aplicação para gerenciamento de eventos de batismo. O objetivo é listar e exibir informações detalhadas sobre cerimônias de batismo, incluindo dados dos batizandos, celebrante e casal responsável pelo batismo.

## Funcionalidades

- Listagem de todos os batismos cadastrados
- Visualização dos detalhes de cada batismo, como data, horário, batizandos, celebrante e casal responsável
- Tela de login para acesso restrito

## Tecnologias Utilizadas

- **Next.js** (React, SSR/SSG)
- **TypeScript**
- **Tailwind CSS** (via PostCSS)
- **React Icons**
